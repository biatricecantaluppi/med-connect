# 🏥 MedConnect
## 📌 Sobre o projeto
O MedConnect é uma aplicação web voltada para a gestão e conexão entre pacientes e serviços de saúde. A proposta do sistema é centralizar funcionalidades importantes como agendamento, organização de atendimentos e gerenciamento de informações médicas em um único ambiente.

O projeto foi pensado para simular um cenário real de uso, focando na organização do fluxo de atendimento e na melhoria da experiência tanto para usuários quanto para profissionais da área.

## 🎯 Objetivo
Criar uma aplicação completa que represente, na prática, como sistemas modernos de saúde funcionam, integrando frontend e backend em uma arquitetura escalável.

## 🚀 Funcionalidades
Agendamento de consultas
Gerenciamento de usuários (paciente/profissional)
Controle de acesso com autenticação
Organização de dados de atendimento
Comunicação entre frontend e backend via API
Estrutura preparada para expansão
